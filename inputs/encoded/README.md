# Dossier d'encodage

Ce dossier contient les images encodées en base64 après l'exécution du script `encode_images.sh`.

## Contenu généré automatiquement

Les fichiers sont nommés séquentiellement :

```
inputs/encoded/
├── 1.txt
├── 2.txt
├── 3.txt
└── ...
```

Chaque fichier `.txt` contient l'encodage base64 d'une image du dossier `inputs/img/`.

⚠️ Ce dossier est automatiquement créé et rempli par le script. Ne pas modifier manuellement.
