# Dossier d'images sources

Placez vos images (JPG, JPEG) dans ce dossier avant d'exécuter le script `process_images.sh`.

## Formats acceptés

- `.jpg`
- `.jpeg`
- `.JPG`
- `.JPEG`

## Exemple

```
inputs/img/
├── document1.jpg
├── document2.jpg
└── photo_texte.jpeg
```

Le script `encode_images.sh` encodera automatiquement toutes les images de ce dossier en base64.
